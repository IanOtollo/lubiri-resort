## How to Upgrade

### Option 1: Fresh Start (Recommended)
1. Delete your old `lubiri-resort` folder
2. Extract this new ZIP
3. Copy your `.env.local` file to new folder
4. Run:
```bash
npm install
npm run dev
```

### Option 2: Update Existing
1. Backup your current folder
2. Replace these files from the new ZIP:
   - `package.json`
   - `tailwind.config.js`
   - `app/globals.css`
   - `app/page.tsx` (homepage only if you want the new design)
3. Run:
```bash
npm install
npm run dev
```
